# Fase 5 – Parte 1: Machine Learning (previsão de rendimento)

Entrega com **Jupyter Notebook** para análise e previsão de rendimento agrícola (ton/ha) por cultura, com técnicas como PCA, agrupamento (K-Means) e modelos de regressão.

## Estrutura

```
Parte 1/
├── everton_marinhos_rm566767_pbl_fase5.ipynb
├── data/
│   └── crop_yield.csv
└── README.md
```

## Como executar o código

1. Instale as dependências (no mesmo ambiente em que executa o Jupyter):

   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn xgboost
   ```

2. Abra o notebook `everton_marinhos_rm566767_pbl_fase5.ipynb` no Jupyter, VS Code ou outro cliente compatível.

3. Execute as células em ordem. O notebook carrega `data/crop_yield.csv` a partir desta pasta (`Parte 1/`).

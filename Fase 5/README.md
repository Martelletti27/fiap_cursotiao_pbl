## Como executar o código

### Parte 1 - Notebook de Machine Learning

1. Instale as dependências:
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn xgboost
   ```

2. Abra o notebook na pasta `Parte 1`:
   ```
   Parte 1/everton_marinhos_rm566767_pbl_fase5.ipynb
   ```

3. Execute as células em sequência (o notebook carrega `data/crop_yield.csv` do mesmo diretório).

Consulte `Parte 2/comparacao_aws.md` para o comparativo de custos e justificativas de região.
